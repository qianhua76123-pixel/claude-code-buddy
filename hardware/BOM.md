### 4号件详细说明：PCA9685驱动板

```
型号：PCA9685 16路 12bit PWM舵机驱动板
接口：I2C（只占ESP32两个引脚：SDA + SCL）
供电：单独5V供电给舵机（不要用ESP32的3.3V给舵机！）

接线方式：
  ESP32 GPIO8 (SDA) → PCA9685 SDA
  ESP32 GPIO9 (SCL) → PCA9685 SCL
  ESP32 GND         → PCA9685 GND
  电池 5V（通过降压）→ PCA9685 V+

  PCA9685 CH0 → 舵机1（左前腿）
  PCA9685 CH1 → 舵机2（右前腿）
  PCA9685 CH2 → 舵机3（左后腿）
  PCA9685 CH3 → 舵机4（右后腿）

⚠️ 舵机驱动板需要单独供电！
    不能直接用ESP32的电源带4个舵机，电流不够。
    用一个5V 2A的降压模块从锂电池取电。
```
